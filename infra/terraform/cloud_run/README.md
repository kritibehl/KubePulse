# Terraform Cloud Run Skeleton

This folder contains a minimal Terraform skeleton for deploying KubePulse-style release validation services to Cloud Run.

It is intentionally a skeleton, not a production module.

## Resources

- Cloud Run service
- container port configuration
- environment variable placeholder
- service URI output

## Intended Resume Signal

Terraform, Cloud Run deployment structure, cloud deployment readiness, infrastructure-as-code familiarity.
