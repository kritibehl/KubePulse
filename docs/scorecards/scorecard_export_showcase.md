# Scorecard Export Showcase

KubePulse scorecards summarize whether a backend or AI service is actually safe to operate after disruption.

## Scorecard highlights

- scenario
- recovery time
- availability achieved
- p95 / p99 latency
- SLO pass/fail
- error budget remaining
- fallback success rate
- degraded-serving mode vs full outage
- probe false positive
- rollout / remediation interpretation
