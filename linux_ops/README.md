# Linux Ops Scripts

Operational scripts for KubePulse service-health troubleshooting and incident capture.

## Scripts

| Script | Purpose |
|---|---|
| `disk_health_check.sh` | Checks disk usage against threshold |
| `network_diag.sh` | Runs ping, port, and HTTP diagnostics |
| `service_restart_guard.sh` | Guards against unsafe service restarts |
| `incident_snapshot.sh` | Captures system, process, port, health, and metrics snapshots |

These scripts make KubePulse more realistic for SRE, production engineering, DevOps, and infrastructure operations workflows.
